## Postwoman

Postwoman é uma plataforma de APIs 100% CLI.

### Como instalar
- Clone o repositório
- Deixe seu terminal no diretório do projeto
- Instale [Ruby](https://www.ruby-lang.org/en/documentation/installation/) e Bundle na sua máquina
- `$ cp .env{.example,}`
- `$ bundle install`
- `$ ./postwoman`
